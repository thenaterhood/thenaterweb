        Manages conditionally redirecting pages based on the requested uri and a given uri to redirect from.

        Checks to see if the uri requested by the user contains the uri to redirect from, and prepares to perform the redirect if so.

        Relies on the redirect class to manage the actual redirecting, this class is primarly an interface to it that deals with the conditional aspect.
