condRedirect¶

Manages conditionally redirecting pages based on the requested uri and a given uri to redirect from.

Checks to see if the uri requested by the user contains the uri to redirect from, and prepares to perform the redirect if so. Relies on the redirect class to manage the actual redirecting, this class is primarly an interface to it that deals with the conditional aspect.
« More »

sanitation¶

Manages sanitizing user input. Currently works only for strings, but adding functions to sanitize other data types is trivial.

Retrieving sanitized values from the class involves invoking the __get function with the type, str, arr, bool, etc for whichever sanitation functions are implemented. The sanitization for the requested return is called (returns an empty object if the sanitization can't convert the current one to the requested one).
